# FDM-Printed Subsonic Wind Tunnel — 1:16 Scale Car Aerodynamics

> **FDM 3D Printing · Fluid Mechanics · CAD Design · Experimental Aerodynamics**

A fully functional, FDM-printed subsonic wind tunnel designed and built from scratch to test the aerodynamics of a 1:16 scale die-cast car model. Features a blower-driven flow circuit, honeycomb flow straightener, contraction section, and a custom forked test section mount — all printed in modular sections and assembled without machining.

| Item | Detail |

---

## Key Sections

**Honeycomb Flow Straightener** — Printed hexagonal cell insert that breaks up large-scale turbulence from the blower and aligns airflow before the contraction.

**Contraction Section** — Smooth converging geometry accelerates flow and reduces turbulence intensity. Internal profile designed to avoid boundary layer separation.

**Test Section & Fork Mount** — Forked bracket sized for the 1:16 car model, holding it centered in the flow. Fork geometry minimizes aerodynamic interference.

**Multi-Finger Diffuser Exit** — Decelerates the exit flow and reduces back-pressure on the test section.
